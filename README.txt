
EcoNet — красиво оформленный демо-сайт
--------------------------------------
Файлы:
 - index.html  — главная страница
 - styles.css  — стили (современный дизайн)
 - script.js   — логика демо
 - README.txt  — инструкции

Как запустить:
1) Распакуйте архив.
2) Откройте index.html в браузере.
3) Всё работает локально, без интернета.
4) Для публикации на GitHub Pages:
   - создайте репозиторий
   - добавьте файлы
   - включите GitHub Pages (main branch / root)
   - сайт станет доступен по адресу: https://<логин>.github.io/<репозиторий>
